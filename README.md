# Sales Data Analysis Project

## Overview

This project analyzes raw sales data to uncover business insights through data cleaning, transformation, exploratory data analysis (EDA), and visualization.

## Objectives

* Clean and preprocess raw sales data
* Perform data transformation using SQL and Python
* Conduct exploratory data analysis (EDA)
* Generate visualizations to identify trends, patterns, and customer behavior
* Extract actionable business insights

## Technologies Used

* SQL
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

1. Data Collection

   * Imported raw sales dataset

2. Data Cleaning & Transformation

   * Handled missing values
   * Corrected data types
   * Created new features
   * Performed SQL-based transformations

3. Exploratory Data Analysis (EDA)

   * Sales distribution analysis
   * Customer segmentation analysis
   * Product performance analysis
   * Branch and city-wise analysis

4. Data Visualization

   * Count plots
   * Line plots
   * Distribution plots
   * Correlation analysis
   * Business KPI visualizations

## Key Insights

* Identified top-performing product categories
* Analyzed customer purchasing patterns
* Examined branch-wise sales performance
* Evaluated payment method preferences
* Discovered sales trends across different time periods
